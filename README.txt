Pesquisa, Ordenação e Técnicas de Armazenamento

APS: Métodos de Ordenação

Prof. Emerson Paduan

Especificação do projeto

Este trabalho consistirá em um programa em Java e um relatório de pesquisa e análise.
O programa Java deve criar vetores aleatórios (usando a biblioteca java.util.Random) de diferentes
tamanhos, ordene estes dados com quatro métodos distintos, contando quantas comparações e
movimentações de valores do vetor foram feitas, e exibindo os totais na tela. Além disse o programa deve
contar quanto tempo cada método levou para ordenar o vetor.
Para realizar a ordenação, o trabalho deverá utilizar três algoritmos: dois algoritmos vistos em sala
(BubbleSort, SelectionSort, InsertionSort, HeapSort ou QuickSort) e dois dos algoritmos abaixo escolhido
pelo grupo:

· MergeSort
· ShellSort
· RadixSort
· BucketSort
· CountSort

Para os algoritmos novos, fazer uma breve descrição sobre a história e características como: se é
estável, in-place, onde se aplica, ou limitações de uso.

Especificação do relatório

O relatório deverá ter 3 partes.

 Na primeira parte o grupo explicará o funcionamento detalhado do algoritmo escolhido na
lista acima, mostrando como ele funciona.

 Na segunda parte, o grupo mostrará os resultados dos testes feitos, com os vetores
gerados e a quantidade de comparações e trocas feitas. Os vetores deverão ter
tamanhos como 1000, 5000, 10.000, 50.000, 100.000, 500.000, 1.000.000. Devem ser
feitos 5 testes para cada algoritmo e tamanho de vetor, com vetores diferentes. Utilize o
valor médio obtido nos testes para montar um gráfico com os resultados obtidos.
ATENÇÃO: lembre-se que o vetor a ser ordenado DEVE ser o MESMO, caso contrário a
comparação dos resultados obtidos na ordenação perde a validade!

 Além de variar os tamanhos do vetor, devem ser feitos estudos sobre o que ocorre se o
vetor de entrada estiver: já ordenado crescente e já ordenado decrescente para estudo de
melhor ou pior caso.

 Na terceira parte, o grupo deve escrever sobre a COMPLEXIDADE teórica do algoritmo e
escrever uma CONCLUSÃO, comparando os dois tipos de ordenação, e quando usar cada
um deles. O grupo não deve apenas repetir as informações obtidas na etapa anterior, mas
analisar os resultados e escrever suas conclusões.

 O grupo deve indicar as fontes usadas como referência na pesquisa.
Informações Gerais

“chegada”, sem repetição de tema entre os grupos. Por tema, entenda-se os algoritmos usados (ou
